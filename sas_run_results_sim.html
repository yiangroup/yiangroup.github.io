LOG
Execution record
The Log is mandatory after every run. Look for NOTE, WARNING, and ERROR. A result can appear even when the log needs review.
OUTPUT
Printed listing
Output is the printed result generated by procedures. Here PROC PRINT and PROC FREQ create listing-style tables.
RESULTS
Navigation tree
Results is a navigation area. It lists generated outputs so you can jump to PRINT or FREQ results. It is not the same as the Log.
